AnswerAI-iOS

Features

Today Page: Displays a collection of featured apps in a scrollable layout.

App Detail Page: Provides detailed information about a selected app.

Custom Animations: Includes smooth transitions and animations for an enhanced user experience.



Technologies Used

SwiftUI: Implemented SwiftUI in order to best desgign common Apple features in a quick way.

Combine: For managing app state and data flow.


How to Run the Project

Clone the repository to your local machine.

Open the .xcodeproj file in Xcode.

Build and run the project on a simulator or physical device running iOS.
Or potentially use SwiftUI's #Preview feature, as shown in AppCardDetailed.swift, to get basic understanding.
